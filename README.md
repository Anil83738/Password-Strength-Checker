# 🔐 Password Strength Checker

A professional cybersecurity mini project developed using Python and Regex to analyze password security strength based on multiple validation rules.

---

## 📌 Project Description

This project checks whether a password is weak, medium, strong, or very strong by validating:

- Minimum password length
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

The application also provides suggestions to improve password security and includes an optional dark-theme GUI using Tkinter.

---

## 🚀 Features

✅ Password strength analysis
